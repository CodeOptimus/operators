# operators


Question 1: List the symbols for each of the below Javascript operators

Arithmetic Operators
Ans: + (addition)
     - (subtraction)
     * (multiplication)
     / (division)
     % (modulo)
     ** (exponentiation)
     ++ (increment)
     -- (decrement)

Assignment Operators
Ans: = (equal to sign)

Comparison Operators
Ans: > (greater than)
     < (less than)
     >= (greater than or equal to)
     <= (less than or equal to)

Logical Operators
Ans: && (AND)
     || (OR)
     ! (NOT)
      
Bitwise Operators
Ans: & (AND)
     | (OR)
     ^ (XOR)
     ~ (NOT)
     << (Zero fill left shift)
     >> (signed right shift)
     >>> (Zero filled right shift)



     
Examples:
'+'; 3+4 = 7
     10+3 =13
'-'; 5-2 = 3
     43-5 = 38
'*'; 3*4 = 12
     5*2 = 10
'/'; 4/2 = 2
     20/5 = 4
'%'; 6%2 = 0
     5&3 = 2
'=' x=y
'>' 5>2
